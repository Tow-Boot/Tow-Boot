Installation instructions

Installing to shared storage

Using the shared storage strategy on the Raspberry Pi Combined AArch64 can be done by writing the shared.disk-image.img to an SD card, USB drive (if the Raspberry Pi is configured correctly) or eMMC (for systems with eMMC).

 # dd if=shared.disk-image.img of=/dev/XXX bs=1M oflag=direct,sync status=progress
