Installation instructions

Installing to shared storage

Using the shared storage strategy on the Olimex TERES-I can be done by writing the shared.disk-image.img to an SD card or eMMC.

 # dd if=shared.disk-image.img of=/dev/XXX bs=1M oflag=direct,sync status=progress
